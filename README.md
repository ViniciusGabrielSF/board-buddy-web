
# BoardBuddy Web

BoardBuddy é uma plataforma que visa conectar uma vasta comunidade de entusiastas de jogos, permitindo que compartilhem e peguem emprestado uma ampla variedade de jogos.

## Instalação

```bash
  yarn install 
```

## Execução

```bash
  yarn start 
```
Acessar: http://localhost:3000/
